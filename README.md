# Demand-Forecasting-and-Inventory-Optimization-for-Milk-Supply-Chain
Demand forecasting and inventory optimization model for milk supply chain using 24 months of demand data (litres). Implements EOQ, safety stock, reorder point, cost comparison, and inventory simulation to reduce total cost by ~20.6% (₹1506 annual savings) while managing demand variability (avg 131.63 litres/month).
